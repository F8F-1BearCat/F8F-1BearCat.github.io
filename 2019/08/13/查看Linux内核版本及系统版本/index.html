查看Linux内核版本及系统版本

发表于 2019-08-13 | 分类于 Linux

简介

记录一下查看Linux内核版本及系统版本的几种命令。

命令

查看Linux内核版本命令：

  • cat /proc/version

  • uname -a

  • uname -sr

查看Linux系统版本的命令：

  • lsb_release -a，列出所有系统版本信息该命令适用于所有的Linux，包括Redhat、SuSE、Debian等发行版。

  • cat /etc/issue，此命令也适用于所有的Linux发行版。

  • cat /etc/redhat-release，该命令只适合Redhat系的Linux。

  • file /bin/bash

  • file /bin/cat
